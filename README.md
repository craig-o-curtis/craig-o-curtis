**What I'm building at [@burglekitt](https://github.com/burglekitt)** — an engineering collective

- **[gmt](https://github.com/burglekitt/gmt)** — *Give Me Temporal!* JavaScript's `Date` is a decades-old bug factory: mutable, timezone-naive, and happy to guess what you meant. GMT wraps the Temporal API in string-in / string-out helpers so date handling stops being a source of production incidents. Nx + pnpm monorepo, published as `@burglekitt/gmt`.
- **[worktree](https://github.com/burglekitt/worktree)** — Git worktrees are excellent and the raw commands are awkward. This CLI turns branch-off → create → copy `.env` → open editor → clean up later into a few commands with sensible prompts. Context switching without the stash-and-pray dance. [Docs](https://burglekitt.github.io/worktree)
- **[run-cv](https://github.com/burglekitt/run-cv)** — a mainframe-inspired terminal CV viewer built with Node, React, and Ink. Markdown-driven, keyboard-navigable, multi-profile. [Landing page](https://burglekitt.github.io/run-cv/)
